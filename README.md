# 基于node+vue的购物商城 购物小程序

## 一、介绍
 基于node+vue的购物商城 购物小程序，分为两个角色，管理员和用户，可以拓展为水果商城，农产品商城等等
 1. 用户：购买商品，领取优惠券，查看订单，添加收货地址等等
 2. 管理员：用户管理，商品管理，订单管理等等
 3. 温馨提示：这个代码不完整，如有需要可咨询我们
## 二、技术栈
1.  前端：vue框架，微信原生小程序
2.  后端：nodejs express框架
3.  数据库：mysql
